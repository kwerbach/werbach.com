Resources and information for Web page designers. Use these materials to learn how to create your own pages, or to improve the pages that you already have.

    The Bare Bones Guide to HTML

      Used by thousands of people around the world, the Guide is a premier resource for Web page designers. It lists virtually every HTML tag in common usage, including the Netscape extensions, in a concise, easy-to-use format. The Guide is now available in nearly 20 different languages!

    What Makes a Good Home Page

      My (brief) thoughts on the stylistic aspects of Web page design.

    HTML FAQ

    WWW Help Page
