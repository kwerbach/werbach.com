Open the random.php3 in a text editor and put in the full system path to the random html directory.2. Upload all the files to that directory.3. Chmod log.txt, show.txt, and settings.php3 to 666.4. Run index.php3 and follow the on screen directions.For further help click on "Help" once you are signed in.If you need a differnt extention other than .php3 just open index.php3 and random.php3 and seach for '.php3' and replace then ALL of them with what you want the extenion to be. Then rename those two files AND settings.php3 with that extention.If you are using this script on your site please contact and tell me because I like to see how my scripts are being used. webmaster@dark-library.comThanks, hope you find this useful~Blake McDonaldwebmaster@dark-library.comGet this script and more at www.phpnewbie.net
