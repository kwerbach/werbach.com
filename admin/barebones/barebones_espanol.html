La ultima version de este documento esta disponible en <http://webarch.com/barebones/>

Esta guía enlista todas las marcas que la mayoria de los visores de Web podrán reconocer. He incluíido todas las marcas en la especificación HTML 3.2 así como las extensiones de Netscape incluidas en versiones de Netscape Navigator hasta la 3.0b5. Esta Gía esta diseñada para ser tan concisa como sea posible; Información mas detallada en HTML se puede encontrar por esta página . Comentarios y sugerencias siempre son bienvenidos.

Indice
  1. MATERIAL INTRODUCTORIO
    • Lo que es Unico acerca de esta guía. (Inglés)
    • Que marcas de HTML estan incluidas? (Inglés)
    • Formato de este documento (incluyendo una descripción de los símbolos y abreviaciones)(Inglés)

  2. MARCAS DE HTML
    • elementos básicos (todos los documentos de HTML deben tener estos)
    • definición estructural (apariencia controlada por las preferencias del browser)
    • formato de presentación (el autor especifica la apariencia del texto)
    • ligas y gráficas
    • divisores
    • listas
    • fondos y colores
    • caracteres especiales
    • formas
    • tablas
    • frames
    • java
    • miscelánea

    Importante: Si no estas seguro acerca de las diferencias entre HTML 2.0, HTML 3.0, HTML 3.2 y las extensiones de Netscape, te sugiero que leas la declaración del W3C en cuanto al desarrollo del HTML.

ELEMENTOS BÁSICOS

 

Tipo de Documento

<HTML></HTML>

(al principio y final del archivo)

 

Titulo

<TITLE></TITLE>

(debe estar en el encabezado)

 

Encabezado

<HEAD></HEAD>

(información descriptiva, tal como el título)

 

Cuerpo

<BODY></BODY>

Contenido de la página.

LIGAS Y GRÁFICAS

 

Ligar algo

<A HREF="URL"></A>

 

 

Ligar a un Ancla

<A HREF="URL#***"></A>

(si el ancla esta en otro documento)

<A HREF="#***"></A>

(si el ancla esta en el mismo documento)

N2.0

Ventana "blanco"

<A HREF="URL" TARGET="***| |_blank|_self|_parent|_top"></A>

 

 

Define un Ancla en un documento

<A NAME="***"></A>

 

 

Relación

<A REL="***"></A>

(no ampliamente implementada)

 

Relación inversa

<A REV="***"></A>

(no ampliamente implementada)

 

Mostrar Imagen

<IMG SRC="URL">

 

 

Alineamiento

<IMG SRC="URL" ALIGN=TOP|BOTTOM|MIDDLE|LEFT|RIGHT>

 

N1.0

Alineamiento

<IMG SRC="URL" ALIGN=TEXTTOP| ABSMIDDLE|BASELINE|ABSBOTTOM>

 

 

Nombre Alternativo

<IMG SRC="URL" ALT="***">

(se muestra si la imagen no esta apagada)

 

Mapa Sensitivo

<IMG SRC="URL" ISMAP>

(requiere un script)

 

Mapa sensitivo del lado del Cliente

<IMG SRC="URL" USEMAP="URL">

 

 

Descripción del mapa

<MAP NAME="***"></MAP>

 

 

Seciones del mapa

<AREA SHAPE="RECT" COORDS=",,," HREF="URL"|NOHREF>

 

 

Dimensiones

<IMG SRC="URL" WIDTH=? HEIGHT=?>

(en pixeles)

 

Reborde

<IMG SRC="URL" BORDER=?>

(en pixeles)

 

Espacio Marginal

<IMG SRC="URL" HSPACE=? VSPACE=?>

(en pixeles)

N1.0

Portal de baja resolución

<IMG SRC="URL" LOWSRC="URL">

 

N1.1

Cliente jala

<META HTTP-EQUIV="Refresh" CONTENT="?; URL=URL">

 

N2.0

Objeto embebido

<EMBED SRC="URL">

(inserta un objeto a la página)

N2.0

Tamaño del objeto

<EMBED SRC="URL" WIDTH=? HEIGHT=?>

<P ALIGN=LEFT|CENTER|RIGHT></P> [*]

(un solo salto de línea)

 

Eliminar Textwrap

<BR CLEAR=LEFT|RIGHT|ALL>

 

 

Barra Horizontal

<HR>

 

 

Alineación

<HR ALIGN=LEFT|RIGHT|CENTER>

<HR SIZE=?>

(en pixeles)

 

Ancho

<HR WIDTH=?>

(en pixeles)

N1.0

Porcentaje de Anchura

<HR WIDTH="%">

Línea Sólida

<HR NOSHADE>

(sin el estilo de chiselado 3D)

N1.0

No Salto de Línea

<NOBR></NOBR>

(evita los saltos de línea)

N1.0

Salto de Palabra

<WBR>

LISTAS

 

 

Lista No Ordenada

<UL><LI></UL>

(<LI> antes de cada elemento de la lista)

 

 

Lista Compacta

<UL COMPACT></UL>

 

 

Tipo de Marca

<UL TYPE=DISC|CIRCLE|SQUARE>

(para todas la lista)

 

<LI TYPE=DISC|CIRCLE|SQUARE>

(esta y subsecuentes)

 

 

Lista Ordenada

<OL><LI></OL>

(<LI> antes de cada elemento de la lista)

 

 

Lista Compacta

<OL COMPACT></OL>

 

 

Tipo de Numeración

<OL TYPE=A|a|I|i|1>

(Para toda la lista)

 

<LI TYPE=A|a|I|i|1>

(esta y subsecuentes)

 

 

Número Inicial

<OL START=?>

(para toda la lista)

 

<LI VALUE=?>

(Esta y subsecuentes)

 

 

Lista de definiciones

<DL><DT><DD></DL>

(<DT>=termino, <DD>=definición)

 

 

Lista Compacta

<DL COMPACT></DL>

 

 

Lista de Menús

<MENU><LI></MENU>

(<LI> antes de cada elemento de la lista)

 

 

Lista compacta

<MENU COMPACT></MENU>

 

 

Lista de Directorios

<DIR><LI></DIR>

(<LI> antes de cada elemento de la lista)

 

 

Lista compacta

<DIR COMPACT></DIR>
