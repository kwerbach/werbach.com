Uses "Internet 2.0" to describe the next-generation internet architecture (not to be confused with the formal Internet2 project) involving ideas like quality-of-service, multicasting, and more robust infrastructure, outlining how the internet's underpinnings might evolve to support new applications beyond the best-effort model of the 1990s.
